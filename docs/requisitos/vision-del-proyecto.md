# Visión del Proyecto

## Nombre del Proyecto

Sistema Web de Gestión de Inventario para el Mini Mercado Ecológico

## Descripción General

El Mini Mercado Ecológico actualmente realiza el control de inventario de manera manual, lo que dificulta el seguimiento preciso de las existencias, entradas y salidas de productos.

Para mejorar este proceso, se desarrollará un sistema web que permita gestionar el inventario de forma organizada y eficiente. Inicialmente el sistema será utilizado para administrar el inventario de maíz, pero se diseñará de forma genérica y escalable para permitir la incorporación de nuevos productos en el futuro.

La solución permitirá registrar movimientos de inventario, controlar existencias, consultar historial de movimientos y generar alertas cuando el stock de un producto alcance niveles mínimos.

## Problema Identificado

La gestión manual del inventario genera riesgos de errores en los registros, pérdida de información y dificultades para conocer la disponibilidad real de los productos.

## Objetivo General

Diseñar y desarrollar un sistema web de gestión de inventario para el Mini Mercado Ecológico que permita registrar entradas y salidas de productos, controlar existencias, consultar movimientos y generar alertas de stock bajo.

## Objetivos Específicos

- Analizar los procesos actuales de control de inventario de la empresa.
- Diseñar una base de datos que permita almacenar la información de productos y movimientos.
- Implementar un sistema de autenticación para los usuarios.
- Desarrollar funcionalidades para registrar entradas y salidas de inventario.
- Implementar consultas y reportes básicos sobre el estado del inventario.
- Generar alertas cuando el stock de un producto alcance niveles mínimos.

## Alcance

El sistema permitirá:

- Gestionar productos.
- Registrar entradas de inventario.
- Registrar salidas de inventario.
- Consultar existencias disponibles.
- Consultar historial de movimientos.
- Gestionar usuarios.
- Visualizar alertas de stock bajo.
- Visualizar indicadores básicos en un dashboard.

## Beneficios Esperados

- Mejor control de las existencias de productos.
- Reducción de errores asociados a registros manuales.
- Disponibilidad de información actualizada.
- Optimización del proceso de gestión de inventario.
- Mejor seguimiento de entradas y salidas de productos.
- Facilitar la toma de decisiones mediante información organizada y confiable.
