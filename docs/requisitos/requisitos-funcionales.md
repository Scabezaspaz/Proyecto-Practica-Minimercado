# Requisitos Funcionales

## RF01 - Autenticación de usuarios

El sistema deberá permitir que los usuarios autorizados inicien sesión mediante credenciales válidas.

## RF02 - Gestión de usuarios

El sistema deberá permitir registrar, consultar y administrar usuarios autorizados.

## RF03 - Registro de productos

El sistema deberá permitir registrar productos dentro del inventario.

## RF04 - Edición de productos

El sistema deberá permitir modificar la información de los productos registrados.

## RF05 - Consulta de productos

El sistema deberá permitir visualizar la información de los productos registrados.

## RF06 - Registro de entradas de inventario

El sistema deberá permitir registrar el ingreso de existencias para un producto.

## RF07 - Registro de salidas de inventario

El sistema deberá permitir registrar la salida de existencias para un producto.

## RF08 - Actualización automática de stock

El sistema deberá actualizar automáticamente las existencias después de cada movimiento.

## RF09 - Consulta de movimientos

El sistema deberá permitir consultar el historial de movimientos realizados.

## RF10 - Configuración de stock mínimo

El sistema deberá permitir establecer una cantidad mínima de existencias por producto.

## RF11 - Generación de alertas

El sistema deberá generar alertas cuando el stock de un producto sea igual o inferior al stock mínimo definido.

## RF12 - Dashboard principal

El sistema deberá mostrar información general sobre el estado del inventario.
