# Requisitos No Funcionales

## RNF01 - Seguridad

El sistema deberá requerir autenticación para acceder a las funcionalidades protegidas.

## RNF02 - Usabilidad

La interfaz deberá ser intuitiva y fácil de utilizar.

## RNF03 - Rendimiento

Las consultas y operaciones deberán responder en tiempos adecuados para el usuario.

## RNF04 - Escalabilidad

La arquitectura deberá permitir incorporar nuevos productos sin modificaciones significativas.

## RNF05 - Disponibilidad

El sistema deberá estar disponible durante el horario de operación de la empresa.

## RNF06 - Mantenibilidad

El código deberá estar organizado y documentado para facilitar futuras mejoras.

## RNF07 - Compatibilidad

El sistema deberá funcionar correctamente en navegadores modernos.

## RNF08 - Integridad de datos

La información almacenada deberá mantenerse consistente y libre de duplicidades.
